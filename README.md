## Development requirements

* Node.JS
* npm
* yarn (Opitional)

  

## Install dependences of project
Using yarn repository:
```
yarn install
```
Using npm repository:
```
npm install
```
## Starting the project
Using yarn:
```
yarn start
```
Using npm:
```
npm start
```
